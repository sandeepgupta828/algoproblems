How to run the program?

After unzipping.

(1)
Make sure you have gradle installed
https://gradle.org/install/

(2)
Make sure you have java installed

(3)
Go in the directory and build
./gradlew build

(4)
Run with arguments:
java -jar build/libs/algoproblems-1.0-SNAPSHOT.jar /Users/new/IdeaProjects/algoproblems/sample.csv 2019-01-01
